Manual ‚Äî Liga√ß√£o Node.js + MongoDB Passo a Passo

1

2

3

1. Cria conta em mongodb.com/atlas ‚Üí Cria cluster gratuito (M0)
2. Em Database Access ‚Üí cria utilizador com password
3. Em Network Access ‚Üí adiciona IP 0.0.0.0/0 (ou o teu IP)
4. Em Connect ‚Üí copia a connection string

# .env
MONGODB_URI=mongodb+srv://utilizador:password@cluster.mongodb.net/minha_bd
4

  .then(() => console.log('‚úÖ MongoDB ligado'))
  .catch(err => console.error(err));

  res.json({ status: 'ok' }));
