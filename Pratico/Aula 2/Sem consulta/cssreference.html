CSS Básico

1. O que é CSS e como ligar ao HTML?

CSS (Cascading Style Sheets) é a linguagem usada para definir o aspeto visual de uma página HTML. Para ligar um ficheiro CSS externo, coloca a tag <link> dentro do <head>:

<!-- Dentro do <head> do HTML --> <link rel="stylesheet" href="style.css">

⚠️ Esta tag bloqueia a renderização (render-blocking) até o CSS estar carregado — isso é intencional para evitar que a página apareça sem estilos (FOUC).

Onde foi usado: Em todas as páginas deste projeto (index.html, externfiles.html, cssreference.html).

Seletores

2. Seletores CSS

Os seletores indicam a quais elementos HTML as regras CSS se aplicam.

/* Seletor de tag — aplica a TODOS os parágrafos */ p { color: #333; } /* Seletor de classe — aplica a elementos com class="destaque" */ .destaque { background: yellow; } /* Seletor de ID — aplica ao elemento com id="titulo" */ #titulo { font-size: 2rem; } /* Pseudo-classe — aplica quando o rato passa por cima */ a:hover { text-decoration: underline; } /* Pseudo-classe — linhas pares de uma tabela */ tr:nth-child(even) { background: #f0f4f8; }

Onde foi usado: nav ul li a:hover, tr:nth-child(even) e form button[type="submit"] no style.css.

Box Model

3. Box Model (Margin, Padding, Border)

Cada elemento HTML é uma "caixa" composta por 4 camadas, de dentro para fora: conteúdo → padding → border → margin.

div { width: 300px; /* largura do conteúdo */ padding: 1rem; /* espaço interior */ border: 2px solid #003366; /* contorno */ margin: 1rem auto; /* espaço exterior + centrar */ box-sizing: border-box; /* padding incluído na largura */ }

Onde foi usado: Em todo o style.css — padding no header, form, table th/td; margin: auto nas imagens e form.

Flexbox

4. Flexbox — Alinhamento e Layout

Flexbox é o sistema de layout CSS para alinhar elementos em linha ou coluna facilmente.

.container { display: flex; /* ativa flexbox */ flex-direction: row; /* direção: linha (padrão) ou column */ justify-content: center; /* alinha no eixo principal */ align-items: center; /* alinha no eixo cruzado */ gap: 1rem; /* espaço entre filhos */ flex-wrap: wrap; /* permite quebra de linha */ }

Onde foi usado: header (logo + título lado a lado), nav ul (links em linha), form (flex-direction: column para campos em coluna).

Tipografia

5. Tipografia (Fontes e Texto)

Para usar fontes do Google Fonts, importa no topo do style.css:

/* Importar fonte do Google Fonts */ @import url('https://fonts.googleapis.com/css2?family=Inter:wght@400;600;700&display=swap'); body { font-family: 'Inter', sans-serif; /* fonte principal */ font-size: 1rem; /* tamanho base (16px) */ font-weight: 400; /* normal=400, bold=700 */ line-height: 1.6; /* espaçamento entre linhas */ color: #1a1a2e; /* cor do texto */ }

Onde foi usado: body no style.css — fonte Inter para todas as páginas do projeto.

Cores

6. Cores e Fundos

O CSS suporta vários formatos de cor:

div { color: #003366; /* hexadecimal */ background-color: rgb(0,51,102); /* RGB */ background-color: rgba(0,51,102,0.8); /* RGBA (com transparência) */ } /* Gradiente linear */ header { background: linear-gradient(135deg, #003366, #0055aa); }

Onde foi usado: header e footer com #003366, fundo do body com #f4f6f9, table th com cor primária.

Efeitos Visuais

7. Sombras e Bordas Arredondadas

Dois dos efeitos mais usados para dar profundidade e modernidade às páginas:

img { border-radius: 10px; /* cantos arredondados */ box-shadow: 0 4px 16px rgba(0,0,0,0.15); /* ^ deslocamento X ^ deslocamento Y ^ desfoque ^ cor com transparência */ } /* Sombra de texto */ h1 { text-shadow: 1px 1px 4px rgba(0,0,0,0.3); }

Onde foi usado: img, iframe, table, form e .ref-section no style.css.

Interatividade

8. Transições e Efeitos de Hover

As transições animam suavemente as mudanças de propriedades CSS:

button { background-color: #003366; transition: background-color 0.2s ease, transform 0.1s ease; /* ^ propriedade ^ duração ^ tipo de animação */ } button:hover { background-color: #0055aa; /* muda cor no hover */ transform: translateY(-1px); /* sobe 1px — efeito de elevação */ }

Onde foi usado: Botão do formulário, links da navegação e linhas da tabela no style.css.

JSON

9. O que é JSON e como se usa em HTML?

JSON (JavaScript Object Notation) é um formato leve de armazenamento e troca de dados. É legível por humanos e facilmente processado por JavaScript. Em HTML, é normalmente carregado via JavaScript com fetch():

// Estrutura de um ficheiro JSON { "nome": "UBI", "ano": 2026, "ativo": true, "paginas": [ { "id": "index", "titulo": "Início" }, { "id": "externfiles", "titulo": "Ficheiros Externos" } ] } // Carregar o JSON com JavaScript (fetch API) fetch('script.json') .then(res => res.json()) .then(data => console.log(data.nome)); // Output: "UBI"

Tipos de dados em JSON:

Tipo Exemplo Notas
string "UBI" Texto entre aspas duplas
number 2026 Inteiro ou decimal, sem aspas
boolean true / false Sem aspas, minúsculas
array ["a", "b"] Lista de valores entre [ ]
object { "chave": "valor" } Pares chave/valor entre { }
null null Valor vazio/ausente

Onde foi usado: script.json neste projeto contém metadados do site, lista de páginas e configurações de estilo.

JavaScript

10. Incluir JavaScript: async vs defer

Tal como o CSS, o JavaScript pode ser ligado externamente. O atributo usado afeta quando o script é executado:

<!-- Bloqueia o parsing do HTML (evitar) --> <script src="script.js"></script> <!-- async: descarrega em paralelo, executa assim que pronto (pode interromper) --> <script src="analytics.js" async></script> <!-- defer: descarrega em paralelo, executa só depois do HTML estar pronto --> <script src="app.js" defer></script>
Atributo Quando executa Quando usar
(nenhum) Imediatamente — bloqueia o HTML Evitar (legado)
async Logo que descarregado Scripts independentes (analytics, ads)
defer Após o HTML estar pronto Scripts que usam o DOM ou dependem de outros

Onde foi usado: <script src="script.js" async> em index.html e externfiles.html.
